Harmony Haven v1.0.6 - Login/Profile Stability Hotfix

แก้ไข:
- ป้องกันการ Redirect วนกลับหน้าเลือกผู้ใช้หลังเข้าสู่ระบบ
- เก็บผู้ใช้ที่เลือกไว้ระหว่างใช้งาน PWA และการเปลี่ยนหน้า
- หลังโหลด Cloud เสร็จ จะไม่เปิดหน้าเลือกผู้ใช้ทับหน้าหลักอีก
- ปุ่มเลือกผู้ใช้เป็น type=button เพื่อไม่ให้ submit ฟอร์มโดยไม่ตั้งใจ
- ถ้า Cloud ชั่วคราวมีปัญหา แอปยังเปิดใช้งานด้วยข้อมูลที่มีอยู่ แทนการค้างหน้าเลือกผู้ใช้
- ตรวจสอบผู้ใช้เดิม หากข้อมูลไม่ถูกต้องจึงค่อยให้เลือกใหม่

วิธีอัปเดต:
1. อัปไฟล์ทั้งหมดทับเวอร์ชัน v1.0.5
2. Redeploy บน Vercel
3. ล้าง Build Cache ในการ Redeploy
4. บนเครื่องผู้ใช้ กด Ctrl+F5 หรือปิด PWA แล้วเปิดใหม่
5. ไม่ต้องรัน SQL เพิ่ม
