Harmony Haven v1.0.2 - Attachment Vault + Card Rename

เพิ่ม:
1) ตั้งค่า > คลังไฟล์แนบ > ดูไฟล์แนบทั้งหมด
2) แสดงชื่อไฟล์ วันที่ ชนิดไฟล์ และขนาดไฟล์
3) ค้นหาและเรียงตามวันที่/ขนาด
4) เปิดไฟล์ผ่าน Signed URL อายุ 5 นาที
5) ก่อนเข้าคลังต้องกรอกรหัสตัวเลข 4-6 หลัก
6) บนคอมพิวเตอร์พิมพ์ตัวเลขจากคีย์บอร์ด กด Backspace และ Enter ได้
7) รหัสเริ่มต้น: 2604 (ควรเปลี่ยนในหน้าตั้งค่าหลังติดตั้ง)
8) บัตรเครดิตมีปุ่ม “แก้ไขชื่อ” โดยรายการเดิมและยอดเดิมไม่หาย

ติดตั้ง:
- อัปโหลดไฟล์ทั้งหมดทับโปรเจกต์เดิม แล้ว Redeploy
- ถ้ายังไม่เคยรัน 002_storage_management.sql ให้รันใน Supabase SQL Editor หนึ่งครั้ง
- ไม่ต้องเปลี่ยน Environment Variables เดิม

หมายเหตุความปลอดภัย:
- รหัสคลังไฟล์เป็นชั้นล็อกในแอปสำหรับผู้ใช้ที่ล็อกอินแล้ว ไม่ใช่การเข้ารหัสไฟล์
- ตัวไฟล์ใน Supabase ยังเปิดผ่าน API ที่ตรวจสอบ session และ Signed URL ชั่วคราว
